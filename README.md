# 🏟️ StadiumGPT

**StadiumGPT** is a next-generation AI-powered Stadium Operations Center and Match Day Companion. Built with React, Vite, and Google's Gemini AI, it transforms the live event experience by providing real-time operational intelligence, dynamic crowd management, and personalized attendee assistance.

---

## ✨ Key Features

### ⚡ Live Stadium Operations
* **AIOperationalInsights**: Real-time analysis of match stages (Pre-match, In-play, Post-match) providing intelligent staff deployment and congestion warnings.
* **Live Event Timeline**: Visual timeline of match day events dynamically synced with current operations.
* **Smart Crowd Heatmap**: Visual representation of crowd density across stadium zones.
* **Auto AI Recommendations**: Proactive suggestions generated by Gemini AI to optimize operations based on live data.

### 🤖 AI Match Day Companion
* **🧭 Smart Route Planner**: Context-aware transit and routing options factoring in current crowd density and weather.
* **🌍 Live Announcement Translator**: Real-time translation of stadium announcements into 7 different languages, powered by GenAI.
* **🌱 Sustainable Match Day**: Interactive gamification encouraging fans to take sustainable actions (e.g., using digital tickets, recycling) and tracking their CO₂ savings.
* **🚨 AI Emergency Assistant**: A floating action button providing immediate access to emergency protocols, evacuation routes, and instant AI triage.
* **📱 Smart Summary**: A dynamic, sticky "Today's AI Match Plan" that collapses intuitively on mobile devices.

## 🚀 Getting Started

### Prerequisites
- Node.js (v20+)
- Google Gemini API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/stadiumgpt.git
   cd stadiumgpt
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

## 🏗️ Architecture & Performance

- **Memoization:** Heavy components like `MatchInfoCard` and `SmartCrowdHeatmap` use `React.memo` to prevent unnecessary re-renders during the 15-second simulation loops.
- **Glassmorphism UI:** Built with custom CSS variables (`--glass-bg`, `--glass-border`) for a highly premium, unified aesthetic without the bloat of external CSS frameworks.
- **Animations:** Smooth, staggered element reveals and cubic-bezier spring modal animations for a highly tactile user experience.
